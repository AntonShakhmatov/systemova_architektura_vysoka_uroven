1-Клиент подаёт онлайн заявку;
2-Система принимает и валидирует данные;
3-Система связывается через API с внешними регистрами и проверяет информацию о клиенте(кредитный регистр, доходы, долги...);
4-Система получает ответ от кредитного регистра(в моём случае CSV файлом)
4-Система рассчитывает кредитное скоринговое значение;
5-Заявка попадает к кредитному специалисту;
6-Специалист принимает решение о выдачи кредита;
7-Система сохраняет результат и посылает клиенту уведомление о решении;
